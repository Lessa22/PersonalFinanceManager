## Personal Finance Manager 

Une application web pour gérer mon budget personnel en Ariary.

## Ce que ça fait

- Suivre mes revenus et dépenses chaque mois
- Planifier mes charges récurrentes (loyer, factures...)
- Fixer des objectifs d'épargne et voir ma progression
- Consulter l'historique de mes transactions avec des filtres

## Technologies utilisées

- HTML, CSS, JavaScript (vanilla, sans framework)
- localStorage pour sauvegarder les données dans le navigateur

## Lancer le projet

1. Cloner le repo
2. Ouvrir le dossier dans VSCode
3. Lancer avec Live Server sur `index.html`
